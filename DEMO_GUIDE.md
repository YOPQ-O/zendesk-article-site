## 📱 確認ポイント

### 1. デバイス別の画面占有率

各ページを以下のデバイスで開いて確認してください：

| デバイス | 確認方法 |
|---------|---------|
| **PC** | デスクトップブラウザで直接アクセス |
| **タブレット** | iPadやAndroidタブレットで開く、またはブラウザのデベロッパーツールでタブレットモードに |
| **スマートフォン** | iPhoneやAndroidスマホで開く、またはブラウザのレスポンシブモードで確認 |

### 2. 表示状態の確認

#### ① 未展開時（ボタン・アイコン状態）
- 右下に表示される**チャットアイコン**のサイズ
- ページコンテンツを隠していないか
- 操作ボタン（スクロール、リンククリック等）の邪魔になっていないか

#### ② 展開時（チャットウィンドウ）
- アイコンをクリックして**チャット画面を開いた状態**
- ウィンドウのサイズ感
- コンテンツの読みやすさ
- ページの他の要素との重なり具合

### 3. 操作性の確認

- ページのスクロールがスムーズか
- リンクやボタンがクリックしやすいか
- ウィジェットが操作の邪魔になっていないか
- モバイルでのタップ操作に支障がないか

## 🎨 ウィジェットのカスタマイズ仕様

### サイズ変更の実装方法

各ページでは、CSSの`transform: scale()`を使用してウィジェットのサイズを調整しています：

```css
/* 小サイズ（75%） */
#launcher {
  transform: scale(0.75) !important;
}

/* デフォルト（100%） */
/* 標準のまま、変更なし */

/* 大サイズ（130%） */
#launcher {
  transform: scale(1.3) !important;
}
```

### カスタマイズ可能な要素

✅ **サイズ調整**: CSS transformで自由に拡大・縮小可能
✅ **配置位置**: 右下、左下、右上、左上など配置変更可能
✅ **オフセット**: 画面端からの距離を調整可能
✅ **表示/非表示**: 特定のページでのみ表示することも可能

## 📝 顧客からの質問への回答例

### Q1: ウィジェットのサイズは自由に変更できますか？

**A:** はい、可能です。CSSを使用して任意のサイズに調整できます。デモページでは75%、100%、130%の3パターンをご用意していますが、他のサイズも実装可能です。

### Q2: ウィジェットの位置は変更できますか？

**A:** はい、可能です。右下（デフォルト）以外にも、左下、右上、左上など、任意の位置に配置できます。

### Q3: 特定のページだけにウィジェットを表示できますか？

**A:** はい、可能です。JavaScriptを使用して、特定のURL、特定のページタイプ、または特定の条件下でのみウィジェットを表示することができます。

### Q4: モバイルとPCで異なるサイズを設定できますか？

**A:** はい、可能です。レスポンシブCSSを使用して、デバイスごとに異なるサイズや配置を設定できます。

### Q5: デモページはいつまで見られますか？

**A:** GitHub Pagesでホストされているため、特別な操作をしない限り永続的にアクセス可能です。無料で、期限もありません。
